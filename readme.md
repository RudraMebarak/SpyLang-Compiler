# SpyLang Compiler

A compiler for a custom detective-themed programming language.

## Features (Planned)
- Handwritten lexical analyzer
- Recursive-descent parser
- Abstract Syntax Tree (AST)
- Symbol table and semantic analysis
- Three-address intermediate code generation

## Tech Stack
- Language: C++
- Lexer: Handwritten
- Parser: Recursive Descent
